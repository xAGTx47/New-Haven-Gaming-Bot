# 🔒 Privacy Policy

_Last updated: March 2026_

This Privacy Policy explains what data the New Haven Gaming Discord bot collects, why it collects it, and how it is used. We take your privacy seriously and only store what is necessary for the Bot to function.

---

## 1. What Data We Collect

The Bot collects and stores only the following data, all within the context of your Discord server:

| Data Type | Purpose |
|---|---|
| Discord User ID | To identify users across economy, rep, pet, and message tracking systems |
| Discord Server (Guild) ID | To keep data isolated per server |
| Economy balances & inventory | To power the coin, shop, and item systems |
| Message count (total & weekly) | To track server activity for the message leaderboard |
| Voice time (total & weekly) | To track time spent in voice channels for the voice leaderboard |
| Pet data | To maintain pet stats, level, and mood between sessions |
| Reputation points | To track rep given between users |
| Invite counts | To track server invites per user |
| Moderation records (warnings) | To log and retrieve user warnings issued by moderators |
| XP & level data | To power the leveling system and rank leaderboard |

---

## 2. What We Do Not Collect

- Message content (beyond counting for activity tracking)
- Usernames, profile pictures, or email addresses
- Any data from direct messages
- Any personally identifiable information beyond Discord IDs

---

## 3. How Data Is Stored

All data is stored in a PostgreSQL database hosted on Neon. Data is not shared with any third parties, sold, or used for advertising purposes. Sensitive operations use atomic database transactions to ensure consistency.

---

## 4. Data Retention

Data is retained as long as the Bot is operational. Economy and activity data may be reset by server administrators at their discretion. If the Bot is removed from a server, data associated with that server is no longer actively used but may remain in storage.

---

## 5. Third-Party Services

The Bot may use the following external APIs to provide certain features:

- **NASA API** — to fetch the Astronomy Picture of the Day for the `/apod` command
- **icanhazdadjoke API** — for the `/dadjoke` command

These services are accessed only when the relevant commands are used. No user data is sent to these services.

---

## 6. Your Rights

You may request deletion of your data at any time by contacting the bot owner through the [New Haven Gaming Discord server](https://discord.gg/QRy4vFh9R5). We will remove your data from all stored records upon a valid request.

---

## 7. Changes to This Policy

This Privacy Policy may be updated at any time. Continued use of the Bot following any changes constitutes acceptance of the updated policy.

---

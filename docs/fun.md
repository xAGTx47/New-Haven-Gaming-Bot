# 🎉 Fun

![Fun Banner](.gitbook/assets/fun-banner.png)

Lighthearted commands for entertainment.

***

| Command                 | Description                                                                                              |
| ----------------------- | -------------------------------------------------------------------------------------------------------- |
| `/8ball <question>`     | Ask the magic 8-ball a yes/no question — it knows all                                                    |
| `/dice [sides] [count]` | Roll one or more dice. Defaults to a standard 6-sided die                                                |
| `/rps <choice>`         | Play rock, paper, scissors against the bot                                                               |
| `/joke`                 | Get a random joke                                                                                        |
| `/dadjoke`              | Get a random dad joke                                                                                    |
| `/trivia`               | Answer a random trivia question — tests your general knowledge                                           |
| `/guess [max]`          | Start a number guessing game in the channel — members type their guesses and the bot responds with hints |
| `/rickroll <user>`      | Send a rick roll to an unsuspecting member                                                               |
| `/roast <user>`         | Hit someone with a brutal one-liner roast                                                                |
| `/wyr`                  | Get a Would You Rather question — vote with the buttons on the embed                                     |
| `/pp [user]`            | The classic pp size check                                                                                |

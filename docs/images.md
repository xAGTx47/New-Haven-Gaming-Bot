# 🖼️ Image Commands

GIF reactions and image utilities.

---

## NASA

| Command | Description |
|---|---|
| `/apod` | Get NASA's Astronomy Picture of the Day — a new image every day |

## Reactions

Use these to express yourself or interact with other members.

| Command | Target Required | Description |
|---|---|---|
| `/boop <user>` | Yes | Boop someone on the nose |
| `/hug <user>` | Yes | Give someone a warm hug |
| `/kiss <user>` | Yes | Kiss someone |
| `/poke <user>` | Yes | Poke someone |
| `/slap <user>` | Yes | Slap someone |
| `/wave <user>` | Yes | Wave at someone |
| `/bored` | No | Express your boredom with a GIF |
| `/clap` | No | Clap it up |
| `/dance` | No | Show off your moves |
| `/happy` | No | Share some happiness |
| `/sad` | No | Express your sadness |
| `/tired` | No | Show how tired you are |
